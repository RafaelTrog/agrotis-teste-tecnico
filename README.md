
## Teste técnico front-end (SAP)

Formulário simples seguindo protótipo e regras recebidas.


### Tecnologias Utilizadas
- React
- Vite
- Material-UI
- React Hook Form
- Zod
- Styled Components
- Typescript


### Instalação e Configuração (Comandos)
#### Instalação de dependências:
```
yarn
```
ou
```
npm i
```

#### Execução (localhost:3000)
```
yarn dev
```
ou
```
npm run dev
```
